# Contributing to Leptos Studio

Terima kasih atas minat Anda untuk berkontribusi!

## Cara Berkontribusi
- Fork repository ini
- Buat branch baru untuk fitur/bugfix
- Lakukan perubahan dan commit
- Buat pull request ke branch `main`

## Pedoman Kode
- Ikuti gaya kode Rust dan Leptos
- Sertakan komentar yang jelas
- Tambahkan/mutakhirkan dokumentasi jika perlu

## Laporan Bug & Saran
- Gunakan Issues di GitLab untuk melaporkan bug atau mengusulkan fitur
